## Packages
framer-motion | Smooth layout transitions and hover effects
recharts | Data visualization for admin dashboard
clsx | Utility for constructing className strings conditionally
tailwind-merge | Utility for merging Tailwind CSS classes safely

## Notes
- RTL Layout: The application uses `dir="rtl"` on the root element for Arabic support.
- Authentication: Uses Replit Auth (`useAuth` hook).
- Images: Uses Unsplash for agricultural placeholder images.
- State Management: Uses React Context for Shopping Cart.
