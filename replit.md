# Al-Qadri Nurseries (مشاتل القادري / مروج لاند)

Full-stack Arabic/English e-commerce platform for an agricultural business.

## Features
- Product catalog with categories
- Nursery gallery with plant-specific categories
- Branches page with Google Maps integration
- Shopping cart and order management
- Admin dashboard for full CRUD operations

## Tech Stack
- Frontend: React, Tailwind CSS, Shadcn UI, Wouter, TanStack Query
- Backend: Node.js, Express
- Database: PostgreSQL (Drizzle ORM)
- Authentication: Custom auth with session support
- Internationalization: i18next (AR/EN)
